# HS300 趋势过滤最佳 SMA 窗口分析 — 执行计划

## 目标

为 V3-B 风险平价策略中 HS300 趋势过滤找到最佳 SMA 窗口参数，平衡 A 股急跌时的信号及时性和正常波动期的假信号频率。

## 候选窗口

20, 30, 50, 60, 75, 100, 120, 150 交易日 SMA

## 分析步骤

### 1. HS300 基本统计
- CAGR、波动率、最大回撤、Sharpe、Calmar — 了解数据特征

### 2. 六大历史大跌事件的信号时效性
对每个事件 + 每个 SMA 窗口，计算：
- 价格跌破 SMA 的日期（信号触发日）
- 从峰值到信号触发日的跌幅（信号滞后损失）
- 延迟天数
- 采用**3天连续确认**规则避免单日毛刺

### 3. 假信号频率统计
- 定义：跌破 SMA 后 10 个交易日内收回，且期间跌幅 < 5%
- 分别统计全历史年均假信号数和排除危机期后的年均假信号数
- 危机期排除：2015-2016股灾+熔断、2018熊市、2020疫情、2022股债双杀

### 4. 极端事件预警能力
- 检查各 SMA 窗口能否在价格跌超 -10% 之前触发信号

### 5. 推荐输出
- Markdown 对比表格
- 推荐窗口参数 + 理由

## 脚本状态

分析脚本已写入 `_analyze_hs300_sma.py`（临时文件，分析后可删除）。

## 输出格式

每窗口一行，各事件信号日/跌幅/延迟、年均假信号数、综合评估。
